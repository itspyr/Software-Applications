# Simple Math Game

A Kotlin-based Android application that generates dynamic arithmetic problems and provides real-time answer validation. This project focuses on clean UI interaction, structured logic, and proper Android app architecture.

---

## Overview

The Simple Math Game challenges users with randomly generated math problems and provides immediate feedback on correctness. The goal of this project was to strengthen Kotlin fundamentals, activity lifecycle management, and interactive mobile UI design.

This app demonstrates structured logic handling, user input validation, and state updates within a responsive Android environment.

---

## Features

- Random arithmetic problem generation  
- Real-time answer validation  
- Interactive user input handling  
- Clean and responsive UI  
- Structured Kotlin logic implementation  
- Activity lifecycle management  

---

## Tech Stack

- Kotlin  
- Android Studio  
- XML Layouts  
- Gradle Build System  
- Android Activity Lifecycle
